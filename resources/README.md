# Resources

This directory contains **curated external resources** that support learning and system building.

Resources are not proofs and not projects.  
They are **inputs**, not outputs.

## What belongs here?
- Books
- Courses
- Papers
- Tools

All resources listed here are:
- intentionally selected
- referenced from learning or proof artifacts when relevant
- not exhaustive, but **signal-based**

## How to use
- Resources should be linked from:
  - `docs/concepts/`
  - `proofs/`
  - `projects/`
- This folder itself does not contain analysis, only organized lists

## Files
- `books.md` – foundational and advanced books
- `courses.md` – structured learning programs
- `papers.md` – research papers and technical articles
- `tools.md` – software, libraries, and platforms
