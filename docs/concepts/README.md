# Concepts

Structured learning notes for foundations (math, ML, systems, evaluation).

## When to write
- After finishing a topic block
- When you can explain it with examples and constraints
- When a concept is used in a proof or project (link it)

## How to write
Use the template:
- `docs/templates/concept-template.md`

File naming:
- `NN-topic-name.md` (e.g., `01-linear-algebra-vectors.md`)

## Quality bar
A concept note must include:
- definition / intuition
- minimal example
- common mistakes
- how it connects to `proofs/` or `projects/`

## Related outputs
- `proofs/02-math/`
- `proofs/03-ml/`
- `roadmap/`
