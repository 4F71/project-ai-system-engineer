# Journal

Weekly logs of progress and evidence.

## When to write
- At the end of each week (minimum 1/week)
- After a milestone or major decision (optional mid-week entry)

## How to write
Use the template:
- `docs/templates/journal-template.md`

File naming:
- `YYYY-MM-WeekNN.md` (e.g., `2026-02-Week02.md`)

## What must be inside
- What I did (deliverables)
- What I learned (specific, not vague)
- Evidence links (paths to `projects/` / `proofs/`)
- Next week focus

## Related outputs
- `projects/`
- `proofs/`
- `roadmap/`
