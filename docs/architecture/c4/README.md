# C4 Diagrams

This folder contains C4-style diagrams:
- System Context
- Containers
- Components (optional)
- Key relationships and boundaries

## When to write
- When a project reaches a clear system boundary
- When explaining architecture to reviewers becomes useful

## How to use
- Keep diagrams simple and readable
- Prefer linking to diagrams from:
  - the project README (`projects/.../README.md`)
  - ADRs (`docs/architecture/adr/`)
