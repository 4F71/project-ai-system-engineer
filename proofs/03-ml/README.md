# Machine Learning Proofs

Proofs demonstrating classical machine learning competence.

## Scope
- Feature engineering
- Model selection
- Evaluation metrics
- Error analysis

## Proof quality bar
A valid ML proof must:
- state the problem
- justify the metric
- analyze results (not just report numbers)
