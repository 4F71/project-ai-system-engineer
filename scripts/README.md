# Scripts

This directory contains **supporting scripts** used to assist learning, experimentation, and system workflows.

Scripts are not products and not proofs.  
They are **utilities** that make other work easier, cleaner, or repeatable.

## What belongs here?
- Small automation helpers
- Data or file manipulation utilities
- Math / ML helper scripts
- One-off tools used across projects or proofs

## What does NOT belong here?
- Core project logic
- Production services
- Long-running systems

## Rules
- Scripts must be simple and readable
- Each subfolder has a clear scope
- Every folder includes a `README.md`

## Related outputs
- `projects/` (where scripts are used)
- `proofs/` (supporting experiments)
- `docs/` (context and explanations)
