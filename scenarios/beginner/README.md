# Beginner Scenarios

Scenarios in this folder focus on **fundamental problem-solving skills**.

## Characteristics
- Narrow scope
- Few constraints
- Single main objective

## When to use
- Early learning stages
- Reinforcing basic concepts
- Translating theory into simple reasoning

## Expected outcome
- Clear problem understanding
- Step-by-step approach
- Identification of missing information
