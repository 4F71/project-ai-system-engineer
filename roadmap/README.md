# Roadmap

This directory defines the **structured learning and growth plan** for becoming an AI System Engineer.

The roadmap is:
- intentional
- phased
- evidence-driven

Progress is validated through `proofs/` and `projects/`, not by completion alone.

## How to read
- Files are ordered by phases and domains
- Each phase builds on the previous one
- Archive contains outdated or superseded plans

## Structure
- `01–06-*.md` – domain-based progression
- `phase-*.md` – system-level growth phases
- `MASTER-ROADMAP.md` – single source of truth

## Rule
A roadmap item is considered **complete** only when:
- a related proof exists in `proofs/`
- or a project implements it in `projects/`

## Related
- `docs/journal/` (weekly tracking)
- `proofs/` (evidence)
- `projects/` (application)
