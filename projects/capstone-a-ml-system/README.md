# Capstone A – ML System

This capstone focuses on building a **classical machine learning system** with clear data flow, evaluation, and system boundaries.

## Scope
- Problem definition
- Data processing
- Model training and evaluation
- System-level reasoning (inputs, outputs, constraints)

## Purpose
To demonstrate the ability to:
- translate a business/problem statement into an ML system
- reason about metrics and trade-offs
- connect models to real usage scenarios

## What belongs here?
- End-to-end ML pipelines
- Evaluation logic
- Documentation explaining decisions and results
