# Inscart – End-to-End ML System

An end-to-end machine learning system built to demonstrate **system thinking over isolated modeling**.

## What this project represents
This project is:
- a consolidation of prior learning
- a system-level implementation
- a foundation reused across other artifacts (e.g. async interviews)

It is **not** a tutorial or toy example.

## System boundaries
- Input: structured data
- Processing: feature engineering + modeling
- Output: evaluated predictions
- Evaluation: explicit metrics and assumptions
